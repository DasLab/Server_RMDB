   Future Development

RMDB version 2.2 BETA, phase III

The following new features is currently under active development, and will be released for preview soon:

  • Interactive data browser:
    For Eterna entries: A new interface with faster search and locate to particular entry or design project, and extract particular entries into separate RDAT file for download.
    For all: use canvas element for faster heatmap rendering.

  • Advanced search: finer and more detailed search tool for all RMDB data, retrieving construct entries instead of whole RDAT.

  • Structure Server: new result page without VARNA applet, enabling boostrapping and result download. Cache modeling run results for all RMDB_ID entries for preview.
