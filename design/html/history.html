   Future Development

RMDB version 2.0 BETA, phase II

The following new features is currently under active development, and will be released for preview soon:

  • Interactive data browser: for general RMDB entries. A new interface powered by JavaScript enabling faster and more responsive data browsing of each RMDB dataset, including plots and channeling to Structure Server.

  • Interactive data browser: for EteRNA entries. A new interface powered by JavaScript, with faster search and locate to particular entry, scrolling of entire EteRNA round RDAT, extracting particular entry into separate RDAT file download, integrating sequence viewer and secondary structure coloring.

  • Advanced search: finer and more detailed search tool for all RMDB data, retrieving construct entries instead of whole RDAT.

  • Structure Server: new result page without VARNA applet, enabling boostrapping and result download.
