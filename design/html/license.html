   Data Archive Usage

Data entries contained in the RMDB archive are under public domain  (free of all copyright restrictions) and are fully available for both non-commercial and commercial use. Users of the data should attribute the original authors listed on the data entry. By using the data in the RMDB archive, the user agrees to abide by the following conditions:

  • The archival data files in the RMDB archive are made freely available to all users. Data files within the archive may be redistributed in original form without restriction. Redistribution of modified data files using the same file name as is in the RMDB database server is prohibited.

  • Data files containing RMDB content may include the RMDB ID entry name (e.g. MDLOOP_STD_0000) only if they are exactly the same as what is residing in the RMDB archive. This does not prevent databases from including RMDB IDs as cross-references where it is clear that they refer to the RMDB archive. The distribution of modified RMDB data in RDAT format and equivalents is not allowed.

  • The user assumes all responsibility for insuring that intellectual property claims associated with any data set deposited in the RMDB archive are honored. It should be understood that the RMDB data files do not contain any information on intellectual property claims.

  • The data in the RMDB archive are provided on an "as is" basis. Resources on this site are provided without warranty of any kind, either expressed or implied. This includes but is not limited to merchantability or fitness for a particular purpose. The institutions managing this site make no representation that these resources will not infringe any patent or other proprietary right.

   Privacy Policy

Standard server log information, such as IP address, time spent on the site, browser type, etc., is collected and assessed on an aggregate, rather than individual, basis in order to track site statistics, data that are most popular to our visitors, performance monitoring, and troubleshooting. Studying access statistics and usage patterns helps to project future hardware needs, and aids in the design of new functionality. We also use this information for site and system security.

When you sign up for a MyRMDB account, we ask you for personal information (such as your name, email address, account password, affiliation). This information is stored on secure servers. We do not share server log and MyRMDB account information with third parties for marketing or other purposes.
